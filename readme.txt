Adds a Media button to the RichText Toolbar in Gutenberg

== Description ==

Adds a Media button to the RichText Toolbar in Gutenberg so that text can be easily linked to files in the media library.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
